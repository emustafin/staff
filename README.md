# my-staff

Персональная команда AI-агентов для управления 
стратегическими целями, задачами и делегированием.

## Как устроен репозиторий

- `agents/` — промпты и конфиги каждого агента
- `knowledge/` — база знаний (цели, команда, контекст)
- `connectors/` — интеграции с внешними системами
- `workflows/` — n8n workflows как код
- `templates/` — шаблоны документов и PDF
- `logs/` — логи улучшений и метрики
